# eBPF go solutions
### Solutions structure
- Solutions have been implemented in respective folders and question numbers have been added at the end of the folder names
- Solutions to first 2 problems require generating go bindings for the eBPF C code (use <code>go generate</code>) to generate go bindinngs
- For building the loader (that loads the eBPF binary), execute - <code>go build -o <"favourite_name_for_binary"></code>
- Usage for binaries is explained in their respective READMEs in the folders.
